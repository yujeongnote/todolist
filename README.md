# Todo List

일정을 그룹별로 관리할 수 있는 심플한 웹 기반 Todo 앱입니다.

## 기능

- **일정 추가** — 제목과 선택적 설명을 입력해 일정 생성
- **완료 처리** — 라디오 버튼 클릭으로 완료 처리, 완료 시각 자동 기록
- **남은 일정 카운터** — 미완료 일정 수 실시간 표시
- **상태 탭 필터** — 전체 / 진행중 / 완료 탭으로 구분
- **그룹 관리** — 원하는 이름으로 그룹 생성·삭제 (예: Home, Work)
- **그룹 지정** — 일정 생성 시 그룹 선택
- **그룹 필터** — 상태 탭과 그룹 필터를 동시에 적용해 원하는 일정만 표시

## 데이터 저장 방식

> **브라우저 메모리(임시 저장)**

데이터는 JavaScript 변수에만 저장됩니다.  
페이지를 새로고침하거나 브라우저 탭을 닫으면 모든 데이터가 초기화됩니다.  
별도 서버나 데이터베이스 연동 없이 단일 HTML 파일로 동작합니다.

## 사용법

`index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 기술 스택

- HTML / CSS / JavaScript (Vanilla)
- 외부 라이브러리 없음
